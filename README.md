# Email Generation Assistant

## Overview
This project generates professional emails based on user intent, key facts, and tone.

## Features
- Email generation using rule-based prompting
- Custom evaluation metrics:
  - Fact Score
  - Tone Score
  - Structure Score
- Model comparison (Advanced vs Basic)
- JSON output of results

## How to Run

1. Open the Jupyter Notebook
2. Run all cells
3. Output:
   - Generated emails
   - Evaluation scores
   - results.json file

## Files
- Email_Assistant.ipynb → main code
- results.json → evaluation output

## Conclusion
The advanced email assistant outperforms the basic model in structure, tone, and readability.
